# Car Rental Management Pipeline

A software engineering project developed to model and manage vehicle rental operations through a structured system development approach. The project combines system analysis, UML modeling, database design concepts, and Java-based implementation to simulate customer registrations, vehicle availability management, and rental processing workflows.

The development process emphasized requirements analysis and system design before implementation. Comprehensive use-case specifications, sequence diagrams, and workflow models were created to define user interactions, system boundaries, and data movement across different operational scenarios.

The system architecture was designed around normalized data structures linking customer information, vehicle records, pricing details, and rental periods. Particular attention was given to maintaining data consistency, minimizing duplication, and supporting efficient management of rental activities.

Custom desktop forms were developed to facilitate user interaction, data entry, and booking management processes. Validation mechanisms and structured error-handling procedures were incorporated to enhance usability and maintain the integrity of application data throughout system operations.

### Technologies Used

* Java
* UML Modeling
* System Analysis and Design
* Database Design Concepts
* Object-Oriented Programming (OOP)

### Concepts Applied

* Requirements Engineering
* Use Case Modeling
* Sequence Diagram Design
* Data Modeling and Normalization
* Input Validation and Error Handling
